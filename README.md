# Australia Living Cost Analysis Project (ABS Data)

## Overview
This project analyses Australian Living Cost Index (LCI) data from 2020 to 2026 using data from the Australian Bureau of Statistics (ABS).

## Objectives
- Analyse living cost trends across Australian household types.
- Identify key contributors to living cost changes.
- Examine the impact of Housing, Food, Health and Transport costs.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualisation
- Trend Analysis

## Key Findings
- Living costs increased across all household types.
- Housing was a major contributor to rising living costs.
- Transport showed the highest volatility during the study period.
- Food costs increased steadily over time.

## Data Source
Australian Bureau of Statistics (ABS) – Selected Living Cost Indexes, Australia.
